# Project Atlas

Project Atlas je modularna AI platforma za prijem upita, prikupljanje lead podataka, zakazivanje i predaju zahteva operateru.

## Trenutni status

- AI Receptionist v1.0.1 je završeno korektivno izdanje, eksportovano, automatski validirano i objavljeno u lokalnom n8n-u.
- Segment 1/8 komercijalnog MASTER ROADMAP-a je završen.
- Komercijalni MVP je u razvoju.
- Sledeći korak je Segment 2/8 — pouzdanost, greške i bezbednost.

## Početne tačke

- `Docs/PROJECT.md` — istorija razvoja i završenih segmenata.
- `Docs/ARCHITECTURE.md` — arhitektura platforme.
- `Docs/master-roadmap.md` — kompletan redosled do komercijalnog MVP-a.
- `Testing/v1.0-live-acceptance.md` — završna prihvatna provera v1.0 i v1.0.1.
- `Workflows/project-atlas-v1.0.1-ai-receptionist.json` — trenutno potvrđeni AI Receptionist workflow.

## Važno

Trenutni v1.0.1 workflow radi u lokalnom n8n okruženju i koristi Google Sheets kao evidenciju leadova. Produkciono postavljanje, stvarno obaveštenje operateru, kalendar, javni chat i WhatsApp obrađuju se redom prema MASTER ROADMAP-u.
